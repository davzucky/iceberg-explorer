## Codebase Patterns
- To run ruff with all dev dependencies, use `uv run --all-extras ruff check src/ tests/`
- PyIceberg's list_namespaces() accepts optional namespace parameter (tuple or string) to list child namespaces
- API uses unit separator (\x1f) for multi-level namespaces in URLs, but PyIceberg uses dot-separated strings
- Conversion needed: ".".join(parent_parts) for parent, ns.split(".") for returned namespaces
- Follow existing patterns: use global singleton pattern for services (get_catalog_service() like get_engine())
- When updating tests, keep existing fixtures if used by other test classes
- Use NoSuchNamespaceError from pyiceberg.exceptions for proper error handling
- Unit separator decoding: use _parse_namespace() helper which splits on \x1f
